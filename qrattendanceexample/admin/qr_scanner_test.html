Generated QR Code

Click "Generate QR Code" to create a test QR code

Test Instructions
  1. Enter a valid student ID (e.g., "2021-001")
  2. Enter a valid event ID (e.g., "1")
  3. Click "Generate QR Code"
  4. Open the admin dashboard QR scanner
  5. Select the same event ID
  6. Start the scanner and point it at the generated QR code
  7. Check the console for debugging information
Note: Make sure the student ID exists in your database and the event ID is valid.
Debugging: Open browser console (F12) to see detailed logs of the scanning process.
