Sidebar Collapse Test

Click the toggle button in the top navigation to collapse/expand the sidebar.

When collapsed, the sidebar will show only icons with tooltips on hover.

Features
  • Sidebar collapses to show only icons
  • Text labels are hidden when collapsed
  • Tooltips show section names on hover
  • Smooth transitions and animations
  • Responsive design
Instructions
  1. Click the hamburger menu button (☰) in the top navigation
  2. Watch the sidebar collapse to show only icons
  3. Hover over the icons to see tooltips
  4. Click the button again to expand the sidebar
